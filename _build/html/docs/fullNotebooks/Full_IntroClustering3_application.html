PCA Dimension reduction#

Sometimes we want to work with a reduced dimension dataset. In these cases we can use different methods, letâ€™s lean how to use principal component analysis (PCA).

This approach will create orthogonal axis that capture the maximum variance in the data.

#load in the PCA
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

# Standardize all the features
scaler = StandardScaler()
X_scaled = scaler.fit_transform(df_cc)

# Create the algorithm
pca = PCA(n_components=2)

# Fit the algorithm
X_pca = pca.fit_transform(X_scaled)

X_pca

Letâ€™s see how much of the variance in the data can be explained by the first two principal components.

# explained variance of each component
explained_var = pca.explained_variance_ratio_
print("Explained variance ratio:", explained_var)

Letâ€™s take a look at what the PCA did.

# Convert it to a dataframe
pca_df = pd.DataFrame(data=X_pca, columns=['PC1', 'PC2'])

# Plot it
sns.scatterplot( data=pca_df, x='PC1', y='PC2', alpha=0.5)
plt.title('Credit Card Data Projected onto First 2 Principal Components')
plt.xlabel(f"PC1 ({pca.explained_variance_ratio_[0]*100:.1f}% var)")
plt.ylabel(f"PC2 ({pca.explained_variance_ratio_[1]*100:.1f}% var)")
plt.grid(True)
plt.show()

Cluster on reduced dimentions#

Next letâ€™s cluster on this reduced space.

from sklearn.cluster import KMeans

# Create the algorithm
kmeans = KMeans(n_clusters=4, random_state=42)

# Fit the algorithm to the data
clusters = kmeans.fit_predict(X_pca)

# Plot the clusters
pca_df['Cluster'] = clusters
sns.scatterplot(x='PC1', y='PC2', hue='Cluster', palette='tab10', data=pca_df)
plt.title('K-Means Clusters in PCA Space')
plt.show()

What do these clusters look like on the original data? Letâ€™s do some summary statistics.

df_cc_k = pd.merge(df_cc, pca_df, left_index=True, right_index=True)
df_cc_k.groupby('Cluster').mean()
