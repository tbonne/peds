This is just to keep notes about the commands for building and publishing this book

1. Make changes on colab

then clone the github pages to get all the new docs (in the colabDocs folder)
git clone https://github.com/tbonne/IntroDataScience.git

3. build book (from one dir above peds)

jupyter-book build mybookname/

2. Git
stage and commit within the peds folder
git add ./*
git commit -m "adding my first book!"
git push

4. publish the book on github pages (from peds folder)

ghp-import -n -p -f _build/html
